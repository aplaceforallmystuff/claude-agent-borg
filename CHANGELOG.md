## [1.0.0] - 2026-01-07

### Added
- Initial release of Borg assimilation agent
- 5-phase systematic assimilation process:
  - Reconnaissance (external system mapping)
  - Gap Analysis (capability comparison)
  - Adaptation Planning (convention mapping)
  - Assimilation (feature integration)
  - Documentation (changelog tracking)
- Infrastructure compatibility checking
- Smart assimilation decision engine (PROCEED/EXTEND/COMPOSE/ITERATE/BLOCK)
- Convention adaptation (TitleCase to kebab-case, etc.)
- Source attribution tracking
- ITERATE mode for pattern extraction without full replacement

### Features
- Analyzes `.claude/` directories for skills, agents, commands, hooks
- Checks dependencies (API keys, MCP servers, CLI tools)
- Generates detailed assimilation reports
- Adapts external conventions to local standards
- Preserves local artifacts (never overwrites without approval)
- Tracks all changes in CHANGELOG-borg.md
